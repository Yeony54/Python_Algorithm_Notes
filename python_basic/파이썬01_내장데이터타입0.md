## Python Built-in Data Type

파이썬에서 기본적으로 많이 쓰는 ```파이썬 내장 데이터 타입``` 정리



> 다른언어에서 Data Structure (자료구조) 라고 부른다.
>
> `int` `str` 등의 data type이 있다.
>
> Python은 data type안에 자료구조를 합쳐 Data Type이라고 부른다.



1. Numeric
   - ```<class int>``` : 정수
   - ```<class float>``` : 실수
2. Sequence
   - ```<class list>``` : `[]`
   - ```<class tuple>``` : `()`
   - `<class range>` : `range(초기값,마지막값,증가값)`
3. Text Sequence
   - `<class str>` : `''`,`""`
4. Mapping
   - `<class dict>` : `{}`
5. Set
   - `<class set>` : `{}`
6. Bool
   - `<class bool>`
